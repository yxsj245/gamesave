# 导入游戏功能

> 自动扫描本地安装的游戏平台，批量导入游戏到 GameSave Manager。

---

## 支持的平台

| 平台 | 检测方式 | exe 自动识别 |
|------|---------|:-----------:|
| Steam | `appmanifest_*.acf` + `libraryfolders.vdf` | ✅ |
| Epic Games | `LauncherInstalled.dat` | ✅ |
| GOG Galaxy | 注册表 `HKLM\SOFTWARE\GOG.com\Games` | ✅（直接读取） |
| Ubisoft Connect | `settings.yaml` | ✅ |
| EA Desktop | `user_*.ini` | ✅ |
| Battle.net | `Battle.net.config` | ✅ |

---

## 使用方法

### 1. 点击「导入游戏」

在主页顶部工具栏，点击「添加游戏」旁边的 **「导入游戏」** 按钮。

### 2. 等待扫描

系统会自动扫描上述 6 个平台的已安装游戏，扫描过程中显示加载提示。

已经添加过的游戏会自动排除，不会重复显示。

### 3. 选择要导入的游戏

扫描完成后，弹出导入窗口，列出所有检测到的游戏：

- 每个游戏默认**折叠**显示，带 **勾选框** 和 **来源平台标签**（如 Steam、Epic 等）
- 顶部有「全选 / 取消全选」快捷操作
- 点击展开后，可以补全以下信息：
  - **游戏名称**：已自动填充，可修改（支持重命名，修改后折叠标题会实时同步更新）
  - **启动进程**：已自动检测（启发式查找游戏 exe），只读显示
  - **游戏存档目录** ⚠️：**必填**，需要手动选择
  - **启动附加参数**：可选
  - **云端服务商**：可选（需要先在设置中配置）
  - **定时备份**：可选，可设置间隔和最大备份数量

### 4. 确认导入

点击「导入所选」，系统会批量添加所有勾选的游戏。

> **注意**：未设置存档目录的游戏会被自动跳过。

---

## 来源标识

导入的游戏会在游戏列表中显示来源平台徽章（如 `Steam`、`Epic` 等），方便区分游戏来源。

手动添加的游戏不显示来源标识。

---

## exe 识别策略

系统采用以下策略自动识别游戏主程序：

1. **GOG**：直接从注册表读取 exe 路径（最准确）
2. **其他平台**：在游戏安装目录及其子目录（`bin`、`Binaries`、`x64` 等）中查找 `.exe` 文件
3. **过滤规则**：排除已知的非游戏 exe（如 `UnityCrashHandler.exe`、`unins000.exe`、`vcredist.exe` 等）
4. **优先策略**：取体积最大的 exe 文件（通常是游戏主程序）

> 启发式识别并非 100% 准确，建议导入后检查启动进程是否正确。如有误，可在编辑游戏中修改。
