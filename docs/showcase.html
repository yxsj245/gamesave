强大的 功能特色

全方位守护你的游戏存档，从手动备份到云端同步一应俱全

🔄

全生命周期自动备份

从启动游戏到退出游戏，自动完成存档备份。支持 Steam Stub 进程检测，告别云存档出错导致的存档损坏问题。

🎯

一键导入平台游戏

支持 Steam、Epic、GOG、Ubisoft、EA、Battle.net 六大平台游戏一键自动导入，无需手动查找存档路径，快速上手。

💾

热备份 · 随时备份

全面支持热备份技术，即使在游戏运行中也能安全备份存档。任何时刻，只要你想，一键即可保存关键进度。

⏱️

智能定时备份

两种定时模式：设定了游戏进程则跟随游戏生命周期自动启停；未设定则手动控制开关。自动轮转管理历史备份。

☁️

云端备份 · 跨设备同步

本地备份完毕后自动上传到云端（阿里云 OSS）。换电脑、硬盘损坏？从云端一键恢复，跨设备存档互通。

🖥️

系统深度集成

系统托盘后台运行、游戏退出后系统通知告知备份结果、支持开机自启动静默模式。全程零干扰。

存档 全生命周期 保护

从启动到退出，每一步都有备份守护

🎮 启动游戏

通过 GSAM 启动游戏，自动校验本地退出存档，将最新安全存档恢复到游戏目录

🔍 进程智能检测

自动捕获游戏 PID，支持 Steam Stub 进程跳转检测，精确追踪真实游戏进程

⏱️ 定时热备份（可选）

游戏运行中按设定间隔自动备份，全程热备份无需退出游戏，定时器跟随进程自动启停

💾 手动热备份（随时）

关键进度随时手动保存，游戏运行中也可安全操作，备份数量无限制

🛑 游戏退出 · 自动备份

检测到游戏进程退出后，自动打包存档快照为退出存档，覆盖上一份

☁️ 自动上传云端

备份完成后如已配置云端，自动同步上传。跨设备互通，硬盘坏了也不怕

🔔 系统通知

全程后台运行，备份结果通过 Windows 原生系统通知告知，零干扰体验
