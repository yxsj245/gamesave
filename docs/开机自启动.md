# 开机自启动

## 功能说明

GameSave Manager 支持开机自启动功能。启用后，系统登录时应用会自动在**后台托盘**启动，不会弹出主窗口界面，不影响正常使用电脑。

用户可以随时通过**双击系统托盘图标**来打开主界面。

## 使用方法

1. 打开 GameSave Manager
2. 进入 **设置** 页面
3. 找到 **启动设置** 卡片
4. 打开 **开机自启动** 开关

关闭开关即可取消开机自启动。

## 技术实现

### 注册表方式

开机自启动通过 Windows 注册表实现，键值路径为：

```
HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
```

注册名称：`GameSaveManager`

### 静默启动参数

启用开机自启动时，注册的启动命令会附带 `--silent` 参数：

```
"C:\path\to\GameSave.exe" --silent
```

当应用检测到 `--silent` 参数时，会在完成初始化后自动隐藏主窗口，仅在系统托盘显示图标。

### 相关文件

| 文件 | 说明 |
|------|------|
| `Services/AutoStartService.cs` | 开机自启动管理服务（注册表读写、参数检测） |
| `Services/ConfigService.cs` | 配置持久化（`AutoStart` 字段） |
| `ViewModels/SettingsViewModel.cs` | 设置页面数据绑定（`IsAutoStartEnabled`） |
| `Views/SettingsPage.xaml` | 设置页面 UI（启动设置卡片） |
| `App.xaml.cs` | 应用启动逻辑（`--silent` 参数检测与窗口隐藏） |

## 注意事项

- 此功能仅在 **Release** 模式下生效（Debug 模式下关闭窗口会直接退出应用，不会最小化到托盘）
- 开机自启动的状态以 Windows 注册表中的实际值为准，设置页面加载时会自动同步
- 无需管理员权限，使用的是 `HKCU`（当前用户）注册表
