# 首次使用欢迎弹窗

## 当前状态

首次启动时的公测欢迎弹窗通知已移除。

应用启动后会直接进入主页面，不再因首次使用而额外弹出 `ContentDialog`。

## 变更说明

本次调整移除了以下内容：

1. `MainPage` 启动时的首次欢迎弹窗显示逻辑。
2. `AppConfig` 中用于记录弹窗状态的 `HasShownWelcome` 配置字段。
3. `ConfigService` 中与欢迎弹窗状态持久化相关的读取和保存方法。

## 配置兼容性

如果旧版本 `config.json` 中仍然保留 `hasShownWelcome` 字段，新版本会忽略该字段，不影响正常启动和使用。
