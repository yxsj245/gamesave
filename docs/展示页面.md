# GameSave Manager 展示页面

## 文件位置

`docs/showcase.html`

## 说明

这是一个用于视频宣传和项目展示的单页 HTML 文件，展示了 GameSave Manager (GSMA) 的核心优势和与旧版 GSM (Electron) 的全面对比。

## 如何使用

直接用浏览器打开 `docs/showcase.html` 即可预览。

> **注意：** 如需显示项目 Logo，请确保 `logo.png` 文件在 `docs/` 目录的上一级目录（即项目根目录）中。

## 页面内容

| 板块 | 说明 |
|------|------|
| 英雄区     | 项目标题、简介和核心数据指标 |
| 功能特色   | 6 大核心功能卡片展示 |
| 技术对比   | GSMA vs GSM 旧版的 14 项维度全面对比表 |
| 原生技术栈 | C#、WinUI 3、.NET 10、MVVM 架构展示 |
| 存档生命周期 | 从启动到云端的完整保护流程时间线 |
| 开源协议对比 | MIT vs GPL-3.0 协议优劣对比 |
| CTA 行动号召 | 下载和查看源码入口 |

## 设计特点

- 深色模式，游戏风格
- 玻璃拟态 (Glassmorphism) 卡片效果
- 渐变背景光球动画
- 滚动渐显 (Intersection Observer) 动画
- 响应式设计，兼容移动端
- 纯 HTML + CSS + JS，无外部依赖框架
