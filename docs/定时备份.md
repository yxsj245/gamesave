# 定时备份功能

## 功能概述

定时备份功能允许用户为每个游戏配置自动定期备份存档，避免忘记手动保存而丢失进度。

## 配置方式

在「添加游戏」或「编辑游戏」对话框中，**定时备份**分组提供以下设置：

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 启用定时备份 | 关闭 | 开关控制是否启用 |
| 备份间隔（分钟） | 30 | 每隔多少分钟执行一次备份，最小 1 分钟 |
| 最大备份数量 | 5 | 超过此数量后自动删除最旧的定时备份 |

## 启停策略

### 有启动进程的游戏

定时备份**跟随游戏生命周期**自动管理：

- 通过应用启动游戏后 → 定时备份自动开始
- 游戏退出后 → 定时备份自动停止

用户无需手动干预。

### 无启动进程的游戏

在游戏列表项右侧显示手动控制按钮：

- 点击 **▶ 按钮** → 启动定时备份
- 运行中显示 **"定时备份中"** 转圈指示 + **✕ 停止按钮**

## 存档管理

- 定时备份产生的存档标签名为「定时备份」
- 超过设定的最大备份数量时，系统自动删除最旧的定时备份
- 退出存档和手动存档不受定时备份数量限制影响
- 定时备份的存档与手动存档一样，支持还原和删除操作

## 注意事项

- 如果上一次定时备份尚未完成，会自动跳过本次（防止并发备份）
- 删除游戏时会自动停止该游戏的定时备份任务
- 编辑游戏修改定时备份参数后，对于无进程游戏运行中的定时任务会自动重启以应用新设置
