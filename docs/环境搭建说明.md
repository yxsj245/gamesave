# GameSave Manager - 环境搭建说明

## 环境要求

| 要求 | 版本 | 说明 |
|------|------|------|
| Windows | 10 1809+ / 11 | WinUI 3 运行环境 |
| .NET SDK | 10.0+ | 已安装 ✅ |
| Visual Studio | 2026 Enterprise | 已安装 ✅（用于调试可选） |

---

## 项目结构

```
gamesave/
├── docs/                         # 文档目录
│   ├── 开发规划.md
│   └── 环境搭建说明.md           # 本文件
└── src/
    └── GameSave/                 # WinUI 3 主项目
        ├── Models/               # 数据模型
        │   ├── Game.cs           # 游戏信息
        │   ├── SaveFile.cs       # 存档备份
        │   └── CloudConfig.cs    # 云存储配置
        ├── Views/                # UI 页面（模板生成）
        ├── ViewModels/           # MVVM 视图模型（模板生成）
        ├── Services/             # 业务服务
        │   ├── IStorageService.cs       # 存储接口
        │   ├── LocalStorageService.cs   # 本地存储
        │   └── CloudStorageService.cs   # 云端存储
        ├── Helpers/              # 工具类（备用）
        ├── Assets/               # 资源文件
        │   └── GameIcons/        # 游戏图标
        ├── App.xaml              # 应用入口
        ├── GameSave.csproj       # 项目文件
        └── GameSave.sln          # 解决方案文件
```

---

## 常用命令

```powershell
# 还原 NuGet 包
dotnet restore src/GameSave/GameSave.csproj

# 编译（Debug）
dotnet build src/GameSave/GameSave.csproj

# 运行应用
dotnet run --project src/GameSave/GameSave.csproj
```

## WinUI 3 CLI 模板

本项目使用以下模板创建，后续可用于新建 WinUI 3 项目：

```powershell
# 已安装模板
dotnet new install VijayAnand.WinUITemplates

# 创建新的 WinUI 3 项目（MVVM 模式）
dotnet new winui -n MyApp -f net10.0 -mvvm
```
